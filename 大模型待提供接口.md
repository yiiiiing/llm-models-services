#### 2. 智能体发送消息
###### 请求参数(暂定)
```agsl
{
    "messages": [
        {
            "role": "assistant", //智能体消息
            "content": "哼"
        },
        {
            "role": "user", //用户消息
            "content": "什么情况？"
        }
    ],
    "scene_id": 73 // 智能体ID
}
```

###### 响应(暂定)
```agsl
{
     "messages": "消息内容"
     "success": "0~100" // 0 失败 1成功
    
}
```
